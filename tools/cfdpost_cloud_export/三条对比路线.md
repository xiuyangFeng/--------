# 给老师看的 CFD vs GNN 三条对比路线

本文是**总览**：三种出图方式各做什么、需要哪些文件、在 CFD-Post 里怎么操作。  
逐步点击路径见 [CFDPOST_操作指南.md](./CFDPOST_操作指南.md)。

---

## 路线总览

| 路线 | 名称 | 数据来源 | 视觉效果 | 适用场景 |
| --- | --- | --- | --- | --- |
| **A** | 原生 CFD 真值 | Fluent `.cas` + **你导出的 `.dat`** | 标准壁面/体网格 Contour，最「正统」 | 展示 CFD 基准解 |
| **B** | 点云 → 面片（插值） | 本工具 CSV → STL 映射 **或** Fluent 内插值 | 光滑三角面云图，接近 CFD 风格 | 论文/答辩主图（GNN 预测） |
| **C** | 点云直接展示 | 本工具 CSV / VTP | 离散点 Contour，能看分布但略「点状」 | 快速核对、误差分布 |

**建议组合（2×2 或 3 列对比）：**

```text
列1: 路线 A — Fluent 原生 WSS / Pressure（.dat）
列2: 路线 B — GNN 预测插值到壁面（STL 或 Fluent 插值后的 .dat）
列3: 路线 C — GNN 点云直接 Contour（可选，作补充）
```

真值若不用 Fluent `.dat`，也可用 CSV 里的 `wss_cfd` / `p_cfd` 走 **路线 B/C**（与 GNN 同点集，对比更公平）。

## 路线 A：原生 CFD 真值（你在 Fluent 中导出 .dat）

### 你需要准备

| 文件 | 来源 | 说明 |
| --- | --- | --- |
| `GUO_XI_JIANG.cas` / `.cas.gz` | 仓库已有 | 网格 |
| `GUO_XI_JIANG.dat` | **你在 Fluent 中 Write Data** | 该时间步流场解 |
| （可选）时间步 | 与 GNN 样本一致 | 如 `merged-1120` ↔ ascii `GUO_XI_JIANG-1120` |

建议本地目录：

```text
tools/cfdpost_cloud_export/fluent_native/
├── GUO_XI_JIANG.cas.gz
└── GUO_XI_JIANG.dat          ← 你从 Fluent 导出后放这里
```

### Fluent 导出 .dat 步骤

1. **File → Read → Case** → `GUO_XI_JIANG.cas.gz`
2. 加载与 GNN 样本 **同一时间步** 的解（瞬态需先跳到对应步，如 1120）
3. **File → Write → Data** → 保存 `GUO_XI_JIANG.dat`（或与 case 同名）
4. 将 `.dat` 拷到 `fluent_native/` 或任意 CFD-Post 可访问路径

### CFD-Post 操作

1. **File → Load Result** → 选 `.cas` + `.dat`
2. **Insert → Location → Wall** → 选壁面边界
3. **Insert → Contour**：
   - WSS：`Wall Shear` / `Skin Friction` / `Wall Shear Stress`（名称因 Fluent 版本而异）
   - 压力：**Volume** 或 **Plane** 截面上的 `Pressure`
4. 截图 → **列 1「CFD 真值」**

### 注意

- 教程里的 **`.cdat`** 多为 CFD-Post 原生或瞬态步文件；Fluent 稳态/单步一般是 **`.cas` + `.dat`**，不要混用扩展名。
- 仓库内 **没有** `.dat`，必须你自己从 Fluent 导出。

## 路线 B：点云 → 面片（插值映射）

GNN 输出是离散点；要得到类似路线 A 的光滑壁面云图，有两条子路径：

### B1 — Python 映射到 STL（推荐，不占用 Fluent 插值）

**一键（在路线 A/B/C 总脚本里已包含）：**

```bash
bash tools/cfdpost_cloud_export/run_all_routes.sh
```

**单独运行：**

```bash
conda activate GNN_vmtk   # 需要 vtk

python tools/cfdpost_cloud_export/map_to_stl_surface.py \
  --csv tools/cfdpost_cloud_export/output/slow__GUO_XI_JIANG__result_features_merged-1120__wall.csv \
  --stl data_new/AG/slow/GUO_XI_JIANG/GUO_XI_JIANG.stl \
  --max-dist 3.0
```

**输出：**

- `output/route_interp/...__stl_mapped.csv` — STL 顶点 + `wss_pred` 等
- `output/route_interp/...__stl_mapped_wall.vtp` — 壁面 WSS 面片云图
- `output/route_interp/...__stl_mapped_interior.vtp` — 腔内压力（映射到 STL 顶点，仅作参考）

**CFD-Post：**

1. 可先 **Load** 原生 `.cas` 作几何背景（透明）
2. **File → Import → VTK** → 选 `*__stl_mapped.vtp`
3. **Contour** → Location 选导入的 VTP → Variable 选 `wss_pred` 或 `wss_cfd`
4. 与路线 A 并排截图 → **列 2「GNN 面片云图」**

`--max-dist`：与 CSV 坐标同单位；`features` 坐标约为 mm 时，常用 **2~5**。

### B2 — Fluent 内 Cloud of Points 插值

**生成 Fluent 专用 CSV：**

```bash
python tools/cfdpost_cloud_export/prepare_fluent_interpolation.py \
  --csv tools/cfdpost_cloud_export/output/slow__GUO_XI_JIANG__result_features_merged-1120__wall.csv
```

**Fluent：**

1. Read Case（`GUO_XI_JIANG.cas.gz`）
2. **Mesh → Interpolate → Cloud of Points**（或 Solution → Interpolation）
3. 读 `route_interp/fluent_cloud/*__fluent_wss_pred.csv`，目标 Zone 选 **壁面**
4. 新变量名如 `wss_gnn`；对 `wss_cfd` 重复（若要用点云真值对照）
5. **File → Write → Data** → `GUO_XI_JIANG_gnn.dat`
6. CFD-Post 加载 case + 新 dat，标准 Wall Contour

**适用：** 必须与 Fluent 体网格完全一致、或需要体内压力场插值时。

## 路线 C：点云直接展示（不插值到面片）

**生成 VTP：**

```bash
python tools/cfdpost_cloud_export/export_pointcloud_vtp.py \
  --csv tools/cfdpost_cloud_export/output/slow__GUO_XI_JIANG__result_features_merged-1120__wall.csv
```

**输出：**

- `output/route_pointcloud/*__wall.vtp` — 全部标量
- `*__wall__cfd.vtp` — 仅真值列
- `*__wall__pred.vtp` — 仅预测列

**CFD-Post 方式 1 — 导入 VTP：**

1. **File → Import → VTK** → `*__wall__pred.vtp`
2. Contour / Point 显示 `wss_pred`

**CFD-Post 方式 2 — CSV Point Cloud：**

1. **File → Import → Point Cloud**（菜单因版本而异）
2. 选 `*__wall.csv`，坐标 `x,y,z`，变量 `wss_pred`
3. 点较密（~1 万壁面点）时尚可；仍不如路线 B 光滑

**适用：** 快速看预测是否合理、看 `err_wss` 空间分布；答辩主图仍建议 **路线 A + B**。

cd /path/to/GNN
bash tools/cfdpost_cloud_export/run_all_routes.sh

## 对比图排版建议

### 壁面 WSS（一行三列）

| CFD 原生 (.dat) | GNN 面片 (路线 B) | GNN 点云 (路线 C，可选) |
| --- | --- | --- |
| 路线 A Contour | `wss_pred` on VTP | `wss_pred` Point Cloud |
| 色标范围一致 | 同左 | 同左 |

### 腔内压力（截面）

- 路线 A：Plane + Fluent `Pressure`
- 路线 B：对 `__interior.csv` 做 Plane 过滤 + 插值（或 Fluent 插 `p_pred` 到体网格再切面）
- 路线 C：`__interior__pred.vtp` 在截面上 Contour

### 公平对比

- **GNN vs CSV 真值**：同点集，用 `wss_pred` vs `wss_cfd`（不必有 .dat）
- **GNN vs Fluent 原生**：路线 A 的 `.dat` vs 路线 B 的 `wss_pred`，色标统一
