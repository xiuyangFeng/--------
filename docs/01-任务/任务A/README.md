## 4. 当前建议的实际用法

- 你正在推进当前 WSS 主线时，只在 `03-V3路线/` 里活动。
- 你正在回看历史 `kNN + GNN`、PINN 阶梯或 Line G/W 时，再进入 `01-V1路线/`。
- 你正在回看 V2P-WSSP、p+WSS 或 WSS loss 历史时，再进入 `02-V2路线/`。
- 你想查“当前结论、当前配置、当前输出目录”时，优先看对应路线 README；跨路线状态再看 `03-共享执行与状态/`。
