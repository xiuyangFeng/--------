> 上位文档：[实验设计总纲](../../实验设计总纲.md) | 执行清单：[任务E血流动力学三维可视化执行清单](./任务E血流动力学三维可视化执行清单.md)

## 1. 规范定位

本规范固定任务 E 所有论文图件的技术标准。图件按三层证据链组织：

| 层级 | 图件编号 | 证明什么 |
|------|----------|----------|
| CFD 真值 | E1–E3 | CFD 场本身的空间分布与几何关系可信 |
| 模型直接输出对比 | E4–E5 | 模型直接输出在空间上对齐 CFD 真值 |
| 后处理指标对比 | E6–E7 | 预测场经后处理后仍能支撑下游结论 |

每张图必须明确标注自己属于哪一层。**禁止将"模型直接输出量"与"后处理衍生指标"混在同一张图中呈现**。

## 3. 图件清单

### Figure E1：CFD 壁面 WSS 真值图（三视角）

**科学问题**：壁面 WSS 空间分布本身是否合理，低 WSS 区域是否对应瘤体关键解剖位置。

**数据来源**：`ascii/` 壁面 `wall-shear` 列（CFD 直接输出，非计算值）

| 参数 | 规定值 |
|------|--------|
| 物理量 | `wall-shear`（CFD 真值，标量） |
| Colormap | `plasma` |
| 值域 | 单病例探索阶段先记录候选值域，多病例横向比较时固定统一值域 |
| 视角 | 正视 / 侧视 / 后视，三视角固定后不再修改 |
| 背景色 | 白色 |
| Colorbar | 下方水平，标题含单位 `(Pa)` |
| 渲染方式 | 壁面点云映射到 STL 表面后渲染（需 `GNN_vmtk` 环境） |

### Figure E2：CFD 内部切面真值图（`|v|` / `p`）

**科学问题**：内部主流区、回流区、瘤体中央切面的流场分布形态。

**数据来源**：`ascii_in/` 内部场数据

| 参数 | 规定值 |
|------|--------|
| 物理量 | `velocity-magnitude` 或 `pressure` |
| `velocity-magnitude` colormap | `viridis` |

| 插值方法 | 优先 Delaunay 三角化 + 重心插值；备选 RBF |
| 轮廓线 | 黑色实线 |

### Figure E3：CFD 三维流线真值图

**科学问题**：瘤内主射流、回流、二次流结构在三维空间的分布。

**数据来源**：`ascii_in/` 体场 + STL 表面

| 参数 | 规定值 |
|------|--------|
| 流线着色量 | `velocity-magnitude` |
| Colormap | `viridis` |
| 表面透明度 | 0.25–0.35 |
| 体场重建 | Delaunay 三角化或从 `.cas` 导入原始网格拓扑 |
| 积分方法 | 四阶 Runge-Kutta（VTK `vtkStreamTracer`） |
| 播种 | 入口截面均匀播种，规则固定后记入配置 |
| 条数 | 待播种规则确定后固定 |
| 环境 | `GNN_vmtk` |

> **注意**：流线图的论文定量价值低于壁面/切面对比图，优先级排在 E1、E2 之后。

## 4. 统一规范

### 4.1 颜色与误差定义

| 类型 | 语义 | 推荐 colormap |
|------|------|---------------|
| 标量真值/预测值（低到高） | 物理量大小 | `plasma` / `viridis` / `RdBu_r` |
| 绝对误差（小到大） | 偏差大小 | `hot_r` |
| 有符号误差（负到正） | 偏差方向 | `bwr` / `coolwarm` |

**硬规则**：
- 绝对误差图 colorbar 从 0 开始，不设对称零值
- 同一物理量的真值图和预测图必须使用完全一致的值域
- 不同病例横向比较时值域必须统一；若仍用病例自适应值域，图注中必须说明

### 4.2 值域管理

| 物理量 | 策略 |
|--------|------|
| `wall-shear` / `wss` | 单病例探索阶段记录候选值域（如 `[0, X] Pa`），多病例时固定统一值域 |
| `pressure` | 若用百分位截断，固定同一统计口径（如训练集 P5/P95），禁止每图自适应 |
| `velocity-magnitude` | 固定统一值域 |
| 周期指标 | 与任务 B 统一 |

### 4.3 图幅与分辨率

| 用途 | DPI | 图幅 |
|------|-----|------|
| 论文正文图 | 300 | 单栏 8.6 cm / 双栏 17.8 cm |
| 汇报 PPT | 150 | 1920×1080 px |

### 4.4 字体与标注

- 无衬线字体（推荐 Arial 或 Helvetica）
- 单位必须写进 colorbar 标题（如 `WSS (Pa)`、`|v| (m/s)`、`p (Pa)`）
- 图题必须含以下关键信息之一：`CFD truth` / `Prediction` / `Derived hemodynamic metric`
- 后处理衍生指标图**必须额外标注** `derived from predicted field`，不得让读者误认为是模型直接输出

## 5. 时间步与周期选择规范

### 5.1 单帧图

- 用于：CFD 真值展示、模型直接输出对比
- 选帧规则必须在日志中记录，候选规则：
  - `|v|` 全场均值峰值
  - 壁面 `wss` 峰值
  - 固定基准帧（如 `step=1120`，即序列首帧）
- 禁止"看起来最好看"的选帧策略

### 5.2 周期图

- `TAWSS / OSI / RRT` 必须使用完整周期（81 帧）
- 若某个预测样本缺帧，只能输出单帧 WSS 图，不能声称为周期指标
- 离散积分公式与任务 B 完全一致

## 6. 与总纲的对应关系

| 图件 | 回答的论文问题 |
|------|----------------|
| E1–E3 | CFD 本身的空间场与几何关系是否可信（方法部分的数据说明） |

## 7. 论文图件实际出图优先级

按论文审稿价值从高到低排列：

| 优先级 | 图件 | 理由 | 当前可行性 |
|--------|------|------|------------|
| 1 | E4-b/c（壁面 WSS 对比） | 审稿人最关注的壁面指标精度 | ⏳ 依赖路线 α 或 β |
| 2 | E1（壁面 WSS 真值） | 证明 CFD 数据本身合理 | ✅ 可做 |
| 3 | E5（内部 `\|v\|/p` 切面对比） | 内部场重建精度的核心证据 | ✅ 可做 |
| 4 | E6–E7（周期指标对比） | 端到端闭环的关键证据 | ⏳ 依赖任务 B |
| 5 | E4-a（壁面压力对比） | 补充壁面精度证据 | ✅ 可做 |
| 6 | E2（内部切面真值） | 辅助说明 CFD 场结构 | ✅ 可做 |
| 7 | E3（流线图） | 视觉补充，定量价值较低 | ⚠️ 技术风险高 |
