# 后处理可视化与插值方法

## 1. 核心问题

外部 baseline 往往只在采样点、surface vertices、2D grid 或 coarse mesh 上输出预测值，而 CFD 真值可能在 Fluent 原始面片/体网格上。为了把预测数据和 CFD 数据公平对比，必须分清：

- **指标口径**：优先在同一批点上逐点比较；
- **展示口径**：可以插值/映射到 STL 或 VTK mesh 上画云图；
- **后处理口径**：WSS/TAWSS/OSI 等梯度和周期指标不能只依赖视觉插值。

## 2. 推荐方法分层

| 方法 | 适用对象 | 优点 | 风险 | 本项目建议 |
| --- | --- | --- | --- | --- |
| 同点逐点对比 | 预测点与标签点一致 | 最公平，不引入插值误差 | 图不够光滑 | 所有数值主表优先采用 |
| 最近邻映射到 STL 顶点 | 壁面点云到表面面片 | 简单、可解释 | 会产生块状/锯齿；对稀疏点敏感 | 可作快速预览，不作最终指标 |
| PyVista `interpolate` | 点云值映射到目标 mesh 点 | 距离加权，适合稀疏点云可视化 | 会平滑极值和热点 | 只用于展示，记录半径/邻居参数 |
| PyVista `sample` / VTK Probe | 从带拓扑 mesh 采样到目标点 | 利用 enclosing cell 插值，mesh 源数据更可靠 | 需要源数据有单元拓扑 | 若可导出 Fluent/VTK 体网格，优先用于切面/流线 |
| ParaView Resample With Dataset | source 数据映射到 destination geometry | 工程软件链成熟，适合批量可视化 | 参数和 tolerance 需记录 | 适合生成论文云图底板 |
| SciPy RBFInterpolator | scattered points 到规则 grid 或面片 | 平滑、易做 2D/3D 插值 | 大点数二次内存，外推风险高 | 只在局部 patch 或少量点中使用，必须限制 neighbors |
| Delaunay + barycentric | 非结构体点到切面/体网格 | 几何含义明确 | 3D 大点云计算重，退化 simplex 风险 | 内部切面真值图可用 |
| normal-line probing | 由近壁速度估 WSS | 符合 WSS 法向梯度定义 | 需要可靠法向、近壁体场、步长 | WSS 后处理优先预研方向 |

**优先级纪律**：如果源数据有真实 cell topology，优先用 `sample` / `ProbeFilter` / `Resample With Dataset` 这类基于单元包围关系的采样；只有当源数据只有离散点云时，才使用距离加权、RBF 或 Delaunay 等 scattered interpolation。

## 3. 对本项目最稳的可视化链

### 3.1 壁面 WSS / TAWSS

1. 指标：在壁面预测点和 CFD 标签点同点比较。
2. 展示：将同点预测值映射到 STL 顶点或 surface mesh。
3. 记录：映射距离分布、插值半径、未命中点比例。
4. 面片统计：若在 STL/VTK 面片上算均值、p95 或区域指标，应优先做面积加权，避免密集采样区域被重复加权。
5. 禁止：只报插值到 STL 后的误差。

### 3.2 内部速度/压力切面

1. 若有原始体网格：用 VTK/ParaView probe/sample 从体网格采样到切面。
2. 若只有点云：用 Delaunay 或局部 RBF 做切面插值。
3. CFD 真值和 Pred 必须用同一切面、同一插值参数。

### 3.3 WSS 从速度场后处理

WSS 是壁面法向速度梯度，不能简单把速度插值到壁面就结束。建议流程：

1. 先做 **CFD velocity oracle**：用 CFD 速度场按同样法向采样/拟合方法反算 WSS，并与 CFD 原生 `wall-shear` 比较；
2. 若 oracle 本身误差过大，说明后处理链不可信，不应把失败归因于模型；
3. oracle 过线后，对每个壁面点建立法向线；
4. 沿法向线在若干距离处采样预测速度和 CFD 速度；
5. 拟合近壁切向速度梯度；
6. 乘以黏度得到 WSS；
7. 与 CFD 原始 WSS 同点比较。

如果外部模型只输出稀疏点云，必须先确认近壁内部点密度是否足够，否则“速度场看起来好”仍可能无法恢复 WSS。

## 4. 必须写进论文/汇报的风险提示

- 插值会平滑局部极值，尤其会抹平 high-WSS hotspot。
- 2D 展开网格上的 R² 不能替代回映射 3D 壁面点后的 R²。
- 视觉云图相似不等于点级误差小。
- 对 WSS/OSI 这种梯度和方向敏感指标，插值误差会被放大。
- CFD 全网格云图与 1.5 万采样点预测云图并排时，必须说明二者的网格/点集不同。
- 欧氏最近邻可能跨越薄壁、分叉或左右支，应加 region、centerline、法向方向或最大距离约束。
- 压力存在参考零点问题，若采用 per-case/per-frame offset correction，必须在指标表中写明。
- 速度向量若经过 PCA 旋转，做流线或 WSS 前必须回到同一物理坐标系。
- 同一结果不要经历双重插值后再与只插值一次的结果比较；必须记录 CFD→target 和 AI→target 的完整映射链。

## 5. 推荐文档记录模板

每次生成可视化图，至少记录：

| 字段 | 内容 |
| --- | --- |
| `source_points` | 预测点/CFD 点来源 |
| `target_geometry` | STL / VTK mesh / 2D grid |
| `method` | nearest / interpolate / sample / RBF / Delaunay |
| `params` | 半径、邻居数、kernel、tolerance |
| `coverage` | 命中率、未命中点数、映射距离分布 |
| `metric_basis` | 指标是否在插值前同点计算 |
| `figure_only` | 是否仅用于展示 |
