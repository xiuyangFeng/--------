# 点云回插面片可视化目录说明（postview）

> 与 Agent 技能 `.cursor/skills/postview-surface-viz/` 配套；方法背景见 [05-点云预测值与真值回插到面片方法](../paper_reproduction/05-点云预测值与真值回插到面片方法.md)。  
> **与** [实验分析对比图目录说明](实验分析对比图目录说明.md) **分工**：后者管 run 级精度柱图/曲线；本文管 **病例级 STL 面片云图 / ParaView 交付包**。

## 1. 口径（必守）

| 目的 | 口径 | 禁止 |
| --- | --- | --- |
| 正式 R² / pooled 指标 | 原始同点点云 CSV 或 run `summary.json` | 在插值面片 VTP 上算 R² 并当主指标 |
| 病例级云图 | 同一 STL + 同一插值法/参数回插 CFD 与 Pred | CFD/Pred 各自缩放色标 |
| 展示帧 | 默认 **`result_features_merged-1146`** | 用 1120 作主汇报帧 |

图注必写：`merged-1146 · t_norm≈0.16 · 收缩期上升段（近似）· Gaussian r=3 mm, sharpness=2`；若报数字须注明 **81 帧 pooled**。

---

## 2. 根目录（两套产物）

### 2.1 ParaView 交付包（推荐）

```
outputs/field/postview/<RUN_TAG>/
```

- `<RUN_TAG>` 示例：`v3p_i6diag_t016_report` · `crown_vp_t016_report` · `crown_pinn_t016_report`
- 批次说明与帧相位：[`outputs/field/postview/README.md`](../../outputs/field/postview/README.md)
- 跨方法对照表：同目录下 `V3P_vs_CROWN_*_对照表_1146.md`

### 2.2 仅三联 PNG（无完整 ParaView 包）

```
outputs/field/plots/stl_surface_compare/<RUN_TAG>/<stem>/
```

- `<stem>`：`{speed}__{CASE}__result_features_merged-1146`
- 中间 CSV/VTP：`tools/cfdpost_cloud_export/output/<RUN_TAG>/route_interp/`

`outputs/` 在 `.gitignore` 中；**路径须写入** [`代码修改与实验推进记录.md`](../02-推进与变更/代码修改与实验推进记录.md) 文首。

## 3. 单病例推荐文件

| 文件 | 内容 |
| --- | --- |
| `<CASE>__surface_wall.vtp` | ★ ParaView 主文件：Gaussian 插值后的 STL 壁面 + 全部标量 |
| `<CASE>__pointcloud_wall.vtp` | 原始壁面点云（对照插值平滑） |
| `<CASE>__mapping_report.json` | 覆盖率、`map_dist`、插值参数 |
| `plots/fig_{wss,p,vel_mag}_triptych.png` | CFD \| Pred \| Error 三联图 |
| `manifest_bundle.json` | run/manifest、插值参数、文件索引 |
| `GNN_blue_white_red.xml` | 蓝-白-红色标 |
| `README_后处理打开说明.md` | ParaView / CFD-Post 操作 |

`_export/*__wall.csv` 为 **指标口径源**；可视化只读其插值结果，不在此重算 pooled R²。

## 4. 生成方式

**环境**：仓库根目录 · **GNN** conda（`export_for_cfdpost` / `map_to_stl_surface` / 三联图）。

### 4.1 GNN / V3P — 完整 ParaView 包

```bash
MANIFEST=outputs/field/<run>/predictions_test_best_wss/manifest.json \
CASE_NAME=slow/GUO_XI_JIANG \
SAMPLE_ID=result_features_merged-1146 \
RUN_TAG=<your_run_tag> \
bash tools/cfdpost_cloud_export/package_postview_case.sh
```

### 4.2 GNN — 仅 WSS/P 三联图

```bash
MANIFEST=outputs/field/<run>/predictions_test_best_wss/manifest.json \
CASE_NAME=slow/GUO_XI_JIANG \
SAMPLE_ID=result_features_merged-1146 \
RUN_TAG=<your_run_tag> \
bash tools/cfdpost_cloud_export/run_case_surface_compare.sh
```

### 4.3 CROWN — 单例 / 三例批量

```bash
# 单例
CROWN_CONFIG=external_baselines/crown_beihang/configs/local/crown_original_vp_split_AG_v1_seed1.json \
CROWN_CKPT=outputs/external_baselines/crown_beihang/<run>/best_model.pt \
CASE_NAME=slow/GUO_XI_JIANG \
SAMPLE_ID=result_features_merged-1146 \
RUN_TAG=crown_vp_t016_report \
bash tools/cfdpost_cloud_export/package_crown_postview_case.sh

# 三例批量
bash tools/cfdpost_cloud_export/run_crown_surface_batch.sh
```

CROWN 指标与 V3P **分表**（见 [外部baseline实验记录规范](外部baseline实验记录规范.md)）；postview 对照表 **不得**写入 V3 母版实验状态表。

### 4.4 默认插值参数

```text
METHOD=gaussian
RADIUS=3.0 mm
SHARPNESS=2.0
MAX_DIST=3.0 mm
FIELD_CMAP=GNN_BWR
```

## 5. 出图与 QC

- **三联图**：CFD \| Pred \| Error；CFD 与 Pred **同一色标**；误差列单独色标
- **出图前**：读 `mapping_report.json`，`valid_ratio` ≥ 95%；STL 与 CSV 无错位
- **Agent 执行**：读 `.cursor/skills/postview-surface-viz/SKILL.md`（细节见同目录 `reference.md`）
