## 2. 逐篇索引

### 2.1 Learning Mesh-Based Simulation with Graph Networks

- 链接：https://arxiv.org/abs/2010.03409
- 优先看：`Table 1` + `Figure 5`
- 图表在看什么：
  - `Table 1` 汇总各数据集的 `1-step prediction error`、`rollout error` 和推理速度
  - `Figure 5(a)` 对比 `Airfoil` 上不同方法的 `Rollout RMSE`
  - `Figure 5(b)` 对比 cloth 场景下不同历史长度/结构时 rollout 是否发散
- 核心结论：
  - 这篇论文最重要的不是单个静态精度，而是 `rollout` 长时稳定性
  - 论文明确写到模型在多个数据集上相对真实求解器可快 `1-2` 个数量级
  - `Figure 5` 是判断它为什么能成为物理仿真 GNN 经典 baseline 的核心证据
- 对你最有用的读法：
  - 如果你要对比 `EPD message passing` 和你的 `Graph Transformer`，先看 `Figure 5` 的 rollout 误差，而不是只看 `1-step` 指标

### 2.2 Physics constrained graph neural network for real time prediction of intracranial aneurysm hemodynamics

- 链接：https://www.nature.com/articles/s41746-026-02404-z
- 优先看：`Table 1` + `Figure 3` + `Table 2`
- 图表在看什么：
  - `Table 1` 给 4 个模型的全局 `1-step RMSE` 和 `50-step RMSE`
  - `Figure 3` 直接展示 `t2` 时刻不同模型的 rollout 质量差异
  - `Table 2` 把误差拆到 `aneurysm bulge / neck / parent artery` 三个解剖区域
- 核心结论：
  - 文中明确指出 `In-PI-MGN` 的 `50-step error` 比原始 `MGN baseline` 低 `6` 倍以上
  - `In-MGN` 和 `In-PI-MGN` 的优势主要体现在 rollout，而不只是 one-step
  - `Figure 8` 显示颈部高流速区域误差大多保持在 `5%` 内，最大误差低于 `8%`
  - `Figure 10` 的 TAWSS 误差全测试集保持在 `10%` 内，局部最大误差约 `9%`
- 对你最有用的读法：
  - 这是你最接近的“血流场 + 物理约束 + 动脉瘤”对照，必须优先看 `50-step RMSE`

### 2.3 MeshMask: Physics-Based Simulations with Masked Graph Neural Networks

- 链接：https://openreview.net/forum?id=bFHR8hNk4I
- 优先看：OpenReview 摘要中的主结果描述；后续如果你要精读 PDF，再补其实验总表
- 图表在看什么：
  - 这篇论文最重要的信息是跨 `7` 个 CFD 数据集的统一 SOTA 结果和掩码预训练消融
- 核心结论：
  - 论文摘要直接给出：对先前最优方法，`long-term prediction accuracy` 提升最高可到 `60%`
  - 预训练时最多随机 mask `40%` 节点
  - 关键价值在于“预训练策略”而不只是某一个 backbone 本体
- 对你最有用的读法：
  - 它更适合作为“增强策略”参照，而不是你任务 A 的第一顺位直接结构 baseline

### 2.4 Learning Hemodynamic Scalar Fields on Coronary Artery Meshes: A Benchmark of Geometric Deep Learning Models

- 链接：https://arxiv.org/abs/2501.09046
- 优先看：`Table 4` + `Table 6` + `Figure 6`
- 图表在看什么：
  - `Table 4`：合成数据集上，不同后端预测 `pressure drop` 的综合指标
  - `Table 6`：患者真实数据上，不同模型的 `test loss / approximation disparity / training cost`
  - `Figure 6`：患者病例上的 vFFR 空间图对比
- 核心结论：
  - 论文全文结论非常明确：在简单几何上多数模型都能做得不错，但在患者级复杂几何上，只有 Transformer 后端保持强泛化
  - 文中指出 `LaB-GATr` 在患者数据上能把压力降误差控制在 `1-10 Pa` 量级
  - 这是“为什么不用 GCN/GAT/PointNet，而选 Transformer”的最直接文献支撑
- 对你最有用的读法：
  - 先看 `Table 6`，它最接近你实际面临的“真实病例异质性”问题

### 2.5 Physics-informed graph neural networks for flow field estimation in carotid arteries

- 链接：https://arxiv.org/abs/2408.07110
- 优先看：`Table 1` + `Figure 6` + `Figure 9`
- 图表在看什么：
  - `Table 1`：`10-fold cross-validation` 下的均值 `±` 标准差，是这篇最关键的稳定性表
  - `Figure 6`：监督损失和物理损失权重变化带来的 boxplot 分布
  - `Figure 9`：Wasserstein distance 的分布和异常值情况
- 核心结论：
  - 这是少数把 `mean ± std`、boxplot 和异常样本分布都给得比较完整的工作
  - 文中明确写到：物理项过强时会出现“回归到平均速度场”的问题
  - 参考点模型的 subject-specific PIGN 对 MRI 的 `Approx. disp.` 为 `0.29 ± 0.14`
- 对你最有用的读法：
  - 如果你要看“方差/稳定性”，这篇比只给主表的论文更有参考价值

### 2.6 Wall Shear Stress Estimation in Abdominal Aortic Aneurysms: Towards Generalisable Neural Surrogate Models

- 链接：https://arxiv.org/abs/2507.22817
- 优先看：`Table 1` + `Table 2` + `Figure 5` + `Figure 8`
- 图表在看什么：
  - `Table 1`：LaB-GATr / LaB-VaTr 的 `mean ± std` WSS/TAWSS/OSI 结果
  - `Table 2`：与 `MultiViewUNet` 等方法用统一 `Snap. NMAE` 的横向比较
  - `Figure 5`：误差如何随 inflow rate 和 AAA 体积变化
  - `Figure 8`：不同网格分辨率下误差分布 boxplot
- 核心结论：
  - 论文明确说 `LaB-GATr` 比 `LaB-VaTr` 的 `MAE` 低约 `0.1 Pa`
  - `Figure 8` 表明不同 mesh resolution 下误差分布差异不显著，`p > 0.05`
  - 纵向病例里 OSI 最大绝对误差约 `0.03`，对应归一化误差约 `6%`
- 对你最有用的读法：
  - 这是任务 B 里最像你自己的直接竞品，优先看 `Table 2` 和 `Figure 8`

### 2.8 A point-cloud deep learning framework for prediction of fluid flow fields on irregular geometries

- 链接：https://geometry.stanford.edu/projects/cfd_pointnet/
- 优先看：项目页摘要 + 原论文主结果图表
- 图表在看什么：
  - 论文核心是“点云直接预测流场”和“跨几何泛化”
- 核心结论：
  - 项目页明确写到：对多目标体和 airfoil 的测试里，模型在保持 `excellent to reasonable accuracy` 的同时，比传统 CFD `快数百倍`
  - 它的价值主要在于说明“不建图”的点云方法能到什么水平
- 对你最有用的读法：

### 2.9 Deep operator learning for blood flow modelling in stenosed vessels

- 链接：https://www.nature.com/articles/s44387-025-00035-5
- 优先看：`Table 1` + `Table 2` + `Figure 3`
- 图表在看什么：
  - `Table 1`：主性能指标
  - `Table 2`：`5-fold cross-validation` 结果
  - `Figure 3`：FFR 的 scatter + Bland-Altman + 按狭窄程度分层误差
- 核心结论：
  - `DeepFFR_MF` 的瞬时压力误差为 `1.2 ± 3.5 mmHg`
  - FFR 差值约 `-0.0029 ± 0.017`，相关性 `r = 0.99`
  - `Figure 3e/f` 说明在 `60%-70%` 狭窄度区间误差更大
- 对你最有用的读法：
  - 这篇更适合回答“为什么不用 Neural Operator/DeepONet”，不适合作为你第一批直接 baseline

### 2.10 Graph Deep Learning for Intracranial Aneurysm Blood Flow Simulation and Risk Assessment

- 链接：https://arxiv.org/abs/2512.09013
- 优先看：`Figure 2/3` + `Figure 13` + 文中主结果段
- 图表在看什么：
  - `Figure 2/3`：in-distribution 和 MATCH 外部分布验证集上的对比
  - `Figure 13`：不同模型大小的 `autoregressive RMSE`
- 核心结论：
  - 作者明确写到：在两个验证集上，相对最强 GNN 对手误差近乎减半
  - `AnXplore` 上平均精度最高可达 `99%`，bulge 内平均 `L2 error = 2.8 ± 0.7 mm/s`
  - `MATCH` 外部分布验证上精度仍高于 `93%`，平均 `L2 error = 5.4 ± 1.1 mm/s`
  - `Figure 13` 显示其新架构在相似训练速度下，RMSE 能做到旧 message-passing SOTA 的约 `1/3`
- 对你最有用的读法：
  - 这篇非常适合支持“Transformer/稀疏注意力路线在动脉瘤时序流场上更强”

### 2.11 Predicting outcomes following endovascular aortoiliac revascularization using machine learning

- 优先看：`Table 2` + `Figure 2` + `Figure 3`
