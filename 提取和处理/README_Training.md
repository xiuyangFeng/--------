# GNN 模型训练指南

本文档介绍了如何使用提取的几何特征数据来训练物理信息图神经网络 (PI-GNN)。

## 📋 流程概览

1.  **数据准备**: 使用 `dataset.py` 将 CSV 点云数据转换为 PyG 图数据格式 (`.pt`)。
2.  **模型定义**: `model.py` 定义了包含二次残差 (QN) 模块的 GNN 架构。
3.  **开始训练**: `train.py` 加载图数据并执行训练循环，最小化数据损失和物理损失 (PINN)。

## 🛠️ 核心模块说明

### 1. 数据加载 (`dataset.py`)
负责将包含几何特征的 CSV 文件转换为 PyTorch Geometric 的 `Data` 对象。
- **输入**: `batch_process.py` 生成的 `.csv` 文件。
- **输出**: `.pt` 文件，包含节点特征 `x`、边索引 `edge_index`、标签 `y` 等。
- **特征**:
    - `Abscissa`, `NormRadius`, `Curvature`, `Tangent` (来自几何提取)
    - `Pos` (归一化坐标)

### 2. 模型架构 (`model.py`)
实现了 **PI-GNN (Physics-Informed GNN)**。
- **Backbone**: 使用 `GATv2Conv` (Graph Attention Network) 进行消息传递。
- **QN 模块**: 引入 `QuadraticResidualBlock`，通过二次项增强模型对非线性流体动力学的拟合能力。
- **输出**: 速度场 `(u, v, w)` 和 压力场 `(p)`。

### 3. 物理约束 (`physics.py`)
实现了 Navier-Stokes 方程的残差计算。
- **动量方程**: 约束流体运动符合牛顿第二定律。
- **连续性方程**: 约束流体质量守恒。
- **无滑移条件**: 约束血管壁面速度为 0。

## 🚀 快速开始

### 第一步：生成图数据

如果你已经有了 `batch_process.py` 生成的 CSV 文件（例如 `result_features_xxx.csv`），运行以下命令将其转换为图数据：

```bash
# 自动查找目录下的 CSV 并转换
python dataset.py
```

这将生成一个 `.pt` 文件（例如 `test_graph.pt`）。

### 第二步：运行训练

确保目录下有 `.pt` 文件，然后运行：

```bash
python train.py
```

脚本会自动：
1.  加载 `.pt` 图数据。
2.  使用几何分块加载器 (GeometricBatchLoader) 处理大规模点云。
3.  在 GPU (如果可用) 上开始训练。
4.  每 10 个 Epoch 保存一次模型权重 (`model_epoch_X.pth`)。

## 📊 监控与调优

- **Batch Size**: 在 `train.py` 中修改 `BATCH_SIZE = 2048`。如果显存不足，请调小此值。
- **Learning Rate**: 在 `train.py` 中修改 `lr=0.001`。
- **物理权重**: 在 `train.py` 的 Loss 计算部分，调整 `loss_phy` 的系数 (默认 0.1)。
