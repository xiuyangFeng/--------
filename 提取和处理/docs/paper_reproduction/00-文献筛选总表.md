## 3. 当前最现实的复现顺序

1. `multiviewunet_aaa_tawss`：验证 2D 展开类方法能否在本项目壁面/TAWSS 上成立。
2. `coronary_mesh_convolution` / `lab_gatr`：优先利用成熟代码建立 artery wall WSS / 等变 surface baseline。
3. `e3_equivariant_aaa_wss`：最直接竞争 baseline，但需确认目标仓库代码完整度。
4. `pointnetcfd`：最快建立点云外部 sanity baseline，检查我们数据是否能被外部点云框架吃进去。
5. `pings` / `meshgraphnet` / `aneug_flow` / `lannelongue_intracranial_pcgcn`：在 mesh 拓扑、数据接口或代码公开情况明确后进入正式复现。
