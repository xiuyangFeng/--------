## 1. 选取原则

本项目的论文核心是：**显式几何先验增强的 GNN 预测 AAA 时空血流动力学场 + 衍生指标恢复 + 临床风险预测（数字孪生闭环）**。

基准模型的选取围绕以下科学问题：

1. **GNN 主干选择的合理性**：为何选 Graph Transformer 而非传统消息传递 GNN？
2. **显式几何先验的增益**：相比无先验的同架构模型，几何特征带来了什么？
3. **与该领域最新 SOTA 方法的对比**：本方法相对于已发表的最佳方法处于什么位置？
4. **非图方法的对照**：点云方法、CNN 方法、Neural Operator 方法能否替代 GNN？

按优先级分为三个梯队：

- **第一梯队**：必须在实验中实现并对比的方法（直接竞争 / 同领域 SOTA）
- **第二梯队**：强推荐引用并讨论、部分可实现对比的方法（相邻领域 / 互补视角）
- **第三梯队**：在 Related Work 或 Discussion 中引用讨论、不要求实验复现的方法

### 2.1 MeshGraphNet

| 字段 | 内容 |
|------|------|
| 论文标题 | Learning Mesh-Based Simulation with Graph Networks |
| 作者 | Tobias Pfaff, Meire Fortunato, Alvaro Sanchez-Gonzalez, Peter W. Battaglia |
| 发表 | ICLR 2021 |
| 链接 | https://arxiv.org/abs/2010.03409 |
| 代码 | NVIDIA PhysicsNeMo 框架已集成；PyG 社区有复现 |

**架构概述**

经典的 Encode-Process-Decode (EPD) 架构。Encoder 将节点/边特征映射到隐空间；Processor 使用多层消息传递 GNN 进行信息聚合；Decoder 将节点嵌入映射回物理量预测。边特征包含相对位置和距离。该架构是物理仿真 GNN 领域引用最多的基准之一。

**与本项目的对比意义**

- 对应实验设计中的 **Baseline A6**
- 回答科学问题：**经典 EPD 消息传递 vs Graph Transformer 注意力机制**
- MeshGraphNet 不使用显式几何先验，不区分节点类型，可以同时验证架构优势和特征工程优势
- 是审稿人最可能要求的文献对照之一

**论文写作中的引用位置**

- **Related Work**：作为"GNN for physical simulation"的奠基工作引用
- **Experiments**：作为任务 A 的实验基线，表中直接报数
- **Discussion**：对比 EPD 架构与 Transformer 的信息传播效率

**BibTeX**

**架构概述**

### 2.3 MeshMask

| 字段 | 内容 |
|------|------|
| 论文标题 | MeshMask: Physics-Based Simulations with Masked Graph Neural Networks |
| 作者 | Paul Garnier, Vincent Lannelongue, Jonathan Viquerat, Elie Hachem |
| 发表 | ICLR 2025 |
| 链接 | https://openreview.net/forum?id=bFHR8hNk4I |
| arXiv | https://arxiv.org/abs/2501.08738 |

**架构概述**

提出掩码预训练策略用于物理仿真 GNN：随机掩掉 40% 的网格节点，强迫模型学习鲁棒的流体动力学表征。采用非对称 Encoder-Decoder 架构和 Gated MLP。在 7 个 CFD 数据集上达到 SOTA，包含一个新的 3D 颅内动脉瘤数据集（每个网格超过 25 万节点）。长期预测精度相较此前最佳方法提升 60%。

**与本项目的对比意义**

- ICLR 2025 **顶会论文**，代表该领域的最新 SOTA

- 掩码预训练是一种**正交的增强策略**，可以探讨是否与显式几何先验叠加
- 可作为"当前最强方法"的对比对象

**论文写作中的引用位置**

- **Experiments**：如果实现其掩码策略作为对比实验（P1 优先级）
- **Discussion**：讨论预训练策略与显式几何先验的互补性

### 2.4 Geometric Deep Learning Benchmark for Hemodynamic Fields

| 字段 | 内容 |
|------|------|
| 论文标题 | Learning Hemodynamic Scalar Fields on Coronary Artery Meshes: A Benchmark of Geometric Deep Learning Models |
| 作者 | Guido Nannini, Julian Suk, Patryk Rygiel, Simone Saitta, Luca Mariani, Riccardo Maragna, Andrea Baggiano, Gianluca Pontone, Jelmer M. Wolterink, Alberto Redaelli |
| 发表 | Computers in Biology and Medicine, Vol. 195, Article 110477 (2025) |
| 链接 | https://arxiv.org/abs/2501.09046 |

**架构概述**

系统性对比了 6 种几何深度学习后端用于冠状动脉网格上的血流动力学标量场预测（虚拟 FFR）：GCN、GAT、GraphSAGE、PointNet、DiffusionNet、Transformer。在 1500 个合成左冠状动脉分叉和 427 个患者级 CFD 仿真上测试。核心结论：**Transformer 后端在患者级异质数据上显著优于其他架构**，是唯一在复杂真实数据上保持强性能的架构。

**与本项目的对比意义**

- 是**几何深度学习在血流动力学预测上的系统性 Benchmark**
- 其核心结论（Transformer > 其他后端）直接支持我们选择 Graph Transformer 的决策
- 可以引用此论文的结论来证明架构选择的合理性
- 其对比的后端（GCN / GAT / GraphSAGE / PointNet 等）与我们实验设计中的 A1-A3 基线高度重合

**论文写作中的引用位置**

- **Related Work**：作为"geometric DL for hemodynamics benchmark"引用，说明 Transformer 已被验证优于其他后端
- **Method**：引用此论文支持我们选择 Graph Transformer 作为主干的决策
