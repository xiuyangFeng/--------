## 评审意见与后续讨论草案（2026-06-11）

**一句话判断**：这份 I 系方案值得保留为"机会池"，但不能直接当作 GPU 执行队列；它的核心价值在于把 V3P 从微调转向"表示、目标、对称性、优化动力学"四类可证伪命题，当前最大风险是机会过多导致归因稀释，必须先用 0 重训 oracle 把 I1/I2/I6/I7/I10/I14/I16 等压缩成 2-3 条真正可训分支。

### A. 当前机会的强点

1. **命题来源不是拍脑袋**：S1 early peak、S2 POD 0.672、S3 θ 周期、S4 G4 内禀标架都来自已有实验或数据结构，比继续堆 loss/head/seed 更有研究价值。
2. **与 G4 换轨自然衔接**：I1/I2/I3/I4/I5/I10/I14 都围绕 `(s,θ)` 规则域展开，能把 G4 Phase 2 的成败变成后续分支选择，而不是孤立实验。
3. **保留了点云主线的诊断出口**：I6/I7 针对 post5463 early-peak 现象，即便 G4 No-Go，也能回答"为什么 best_wss 很早、为什么 p 继续升而 WSS 退化"。
4. **Go/No-Go 意识明显强于旧发散池**：多数方案都写了前置 oracle、单变量对照、分表纪律和泄漏边界，适合拿给其他 AI 做第二轮压力测试。

### B. 关键风险与薄弱假设

1. **最大薄弱假设：POD/重建上限等于可监督泛化上限。** F0 `pod_2d=0.672` 只能说明 GT 场在规范表示下低秩，不说明输入几何/条件足以预测这些系数；I3/I14 必须补"输入到系数/势场的 probe"，否则容易把 target-side oracle 误读成 model-side opportunity。
2. **G4 成功前，I1/I2/I3/I4/I5/I10/I14 的训练价值都依赖展开链路稳定。** 如果 Job 5507 的 Phase 2 低于 0.42 或 2D→3D 反映射抹平 top-k 高 WSS，本方案中大部分 G4 增强只能降级为诊断，不应继续排队训练。
3. **精确增强不等于分布安全。** I1 θ-roll 是规范原点任意时的强机会；I11 镜像在方程上合法，但真实解剖手性可能偏置，必须先做"增强前后几何分布/高 WSS 区域分布"审计，避免创造 off-manifold 样本。
4. **I6 梯度冲突的归因仍可能混淆。** early peak 也可能来自 checkpoint 选择、head 过拟合、WSS loss 噪声或 val 集小样本波动；I6 必须与 I7 同时做，不能只看到 p/WSS 曲线背离就直接上 PCGrad/解耦训练。
5. **I2/I14 的矢量突破门槛应比标量更严格。** `wss_x/y >0.10` 是结构性证据，但还需检查 `τ_s/τ_θ` 反映射后的方向误差、top-k 高 WSS 区域是否正确；否则可能只是坐标变换带来的局部 R² 改善。
6. **I10 近壁壳层张量的物理动机强，但最容易被数据覆盖率卡死。** 若法向插值覆盖率、法向方向一致性、近壁速度质量不过线，训练会把插值噪声结构化放大，风险高于 I1/I2/I6。
7. **方案数量已超过当前管理容量。** I1-I17 若都进入"待跑"，会回到旧问题：多变量堆叠、表格碎片化、成功无法归因。必须把它们分成"本周可做 oracle"、"5507 后训练候选"、"只保留为论文讨论/备选"三层。

### C. 建议修改后的优先级

| 层级 | 建议保留项 | 理由 | 立即产物 |
| --- | --- | --- | --- |
| **P0-诊断必做** | **I6 + I7** | 与 G4 无关，直接解释 post5463 early peak；即使 No-Go 也能形成 training dynamics 证据 | 梯度 cos/范数比日志、ckpt probe 曲线 |
| **P0-G4 门控** | **I1-a/b + I2-o** | G4 Phase 2 若 Go/边际，周期先验和内禀标架是最低成本、最清晰的两条增强 | θ-roll 审计、τ_s/τ_θ POD/probe |
| **P1-目标重参数化** | **I3 或 I14 二选一先做 oracle** | 两者都吃低秩假设；先比较"输入可预测 POD 系数" vs "势场更低频"哪个证据更硬 | train-only POD coefficient probe 或 HH 分解闭合误差 |
| **P1-物理输入** | **I10-o 优先于 I5-c 训练** | 若法向壳层 oracle 不过，近壁差分代理通道也应降权；先验核查比直接加通道更省 | 法向插值覆盖率、GT 速度差分代理相关性 |
| **P2-全局增强/几何** | **I11、I16** | 可能收益稳定，但须先确认解剖分布与变换规则；不应抢 G4/I6 资源 | 宇称表、规范化前后 GBDT probe |
| **暂缓** | **I4/I9/I13/I17/I12/I15** | 架构或组合成本高，依赖前序结果；目前更适合保留为讨论池 | 暂不排训练 |

### D. 推荐最小执行计划（给下一轮 AI 讨论用）

1. **先等 5507，并同时做 0 重训诊断**：I6 Phase 1 插桩方案、I7 checkpoint probe、I2-o、I14-o/I3-o 二选一、I10-o 覆盖率检查。目标是在不占 GPU 的情况下把 I 系从 17 条压缩到 3 条。
2. **如果 5507 Go 或边际**：第一训练批只跑 `G4 plain → I1-a → I1-b`，同 seed 对照；I2-o 过线后再决定是否开 I2/I14，禁止 I1/I2/I3/I5 同时叠加。
3. **如果 5507 No-Go**：G4 训练增强全部暂停，只保留展开失败归因；主线转 I6/I7。若 I7 显示特征被覆写，优先 I6-a 两阶段冻结；若 probe 持平而 val_wss 衰减，优先 head 正则/EMA，而不是 PCGrad。
4. **任何单 seed 正信号只算观察**：进入母版候选前至少补 seed2/3，并做 case bootstrap；尤其是 `wss_x/y` 破 0.10 时必须同步检查病例级 Pa p95、top-k 高 WSS 区域和方向误差。

### E. 建议补写/修订到正文的点

1. 在 I3 增加前置：**输入到 POD 系数的可学性 probe**。只做 GT POD 重建不够，必须用 train 输入特征预测 POD 系数，并在 val/test 上报告系数 R² 与重建场 R²。
2. 在 I14 增加前置：除 HH 闭合误差与谱能量集中度外，补一个"势场系数是否可由输入预测"的 probe，否则只是证明标签更光滑。
3. 在 I11 增加审计：镜像后样本是否偏离真实解剖分布；建议报告入口/出口相对方位、中心线扭转方向、高 WSS 区域位置分布的镜像前后差异。
4. 在 I6 增加反证分支：如果梯度 cos 不负或范数比不失衡，early peak 不得归因于任务冲突，应自动转 I7/head-overfit/val-noise 解释。
5. 在总览表新增"是否依赖 5507 Go"列，把 G4-dependent 与 G4-independent 方案分开，防止 5507 No-Go 后仍误排 G4 增强。
6. 在执行顺序中明确"一次只允许一个训练变量进入主表"：I1/I2/I3/I5/I10/I14 单独过线后才能组合复测，组合只报告系统级最优，不替代归因表。

### F. 供其他 AI 继续讨论的问题

1. **I3 vs I14**：低秩 POD 系数头和 Helmholtz-Hodge 势场重参数化，哪个更可能把 `wss_x/y` 从 0 拉到 >0.10？有没有必要先做一个统一的"目标基底可预测性"评测来裁决？
2. **I1 θ-roll 的严格性**：当前展开链路的 θ=0 原点是否完全任意？是否存在入口截面、分叉参考方向、重力/坐标系或 BC 特征让 θ 原点具有物理含义？
3. **I11 镜像是否会 off-manifold**：血管解剖手性在本数据中是否有稳定偏向？如果有，镜像增强应作为泛化正则还是应关闭？
4. **I6 early peak 的最小证据是什么**：只看 p/WSS 曲线背离是否足够？梯度冲突、head 过拟合、val 噪声三者如何用一个最小实验区分？
5. **G4 No-Go 后是否还保留 I2/I14**：如果 2D magnitude 都学不好，但 τ_s/τ_θ oracle 很强，是否应改做"点云输出内禀分量"而不是继续 2D U-Net？
6. **论文叙事优先级**：如果点级只从 0.425 提到 0.45，但 Pa p95/区域 top-k 明显改善，这是否已经足以作为主叙事？还是必须坚持点级 R² 突破？

## 最终审查结论（2026-06-11）

**最终 verdict**：I 系文档已经覆盖了足够多的潜在突破口，下一步不应继续扩展新想法，而应进入"压缩、排队、立项前证伪"阶段。最终建议只保留 **5 条近期有效动作**：`I6/I7` 训练动力学诊断、`I1` 周期先验、`I2-o` 内禀标架 oracle、`I20-o` 二次流权重可行性统计、`I14-o 或 I3-o` 目标基底可预测性二选一；其余方案暂作观察池，不进入 GPU 队列。

### 1. 总体分类

| 类别 | 方案 | 最终处理 |
| --- | --- | --- |
| **立即做 0 重训诊断** | I6、I7、I2-o、I20-o、I14-o/I3-o | 保留，作为下一轮行动清单 |
| **5507 Go/边际后第一训练批** | I1-a → I1-b | 保留，单变量顺序执行 |
| **5507 Go 后第二梯队** | I2、I14 或 I3、I10-o | 只在前置 oracle 过线后进入 |
| **点云主线备选** | I19、I16、I11 | 暂缓；只有 G4 No-Go 或 I6/I7 指向训练调度/几何分布问题时再开 |
| **长期观察池** | I4、I9、I12、I13、I15、I17、I18 | 不进近期 GPU 队列 |

### 2. 对 I18-I20 的最终判定

**I18 PDE 残差正则化：暂缓，列 P3/理论验证项。** 它听起来最"物理"，但最容易错：壁面局部动量平衡在离散 CFD、展开网格、近壁插值、边界条件缺失时可能并不闭合。若 GT WSS 本身在该近似 PDE 下残差很大，物理 loss 会把模型往错误方向推。I18 只能先做 GT 残差闭合审计，不允许直接加 loss。

**I19 几何复杂度课程学习：保留为 P2 训练调度项。** 它风险低、代码改动可控，但归因较弱：如果有效，很难区分是 curriculum、训练步数、采样分布还是 hard-case oversampling 的贡献。因此它应排在 I6/I7 之后，用于验证"early peak 是否来自复杂样本过早干扰"，不应先于诊断执行。

**I20 Spatial Focal Loss：第三批里最值得保留，升为 P1 oracle。** 它直接瞄准 `τ_θ / wss_x/y` 被主导分量淹没的问题，和 I2 内禀标架、I14 势场重参数化是同一科学瓶颈。前置统计若显示二次流主导区占比很低但对 Pa/top-k 贡献高，则 I20 有立项价值；若该区域本身噪声大或 test 分布不稳定，则关闭。

### 3. 最小可执行路线

1. **不等 5507 也能做**：I7 ckpt probe、I6 梯度 cos/范数插桩方案、I2-o 标架投影统计、I20-o `τ_s/τ_θ` 能量比分布与二次流区域占比。
2. **5507 Go/边际时**：只跑 `G4 plain → I1-a → I1-b`，确认周期先验是否带来可归因增益；同时根据 I2-o 与 I20-o 决定下一条是 I2/I14 还是 I20。
3. **5507 No-Go 时**：G4 增强训练全部暂停；保留 I2-o/I20-o 作为科学诊断，不再上 I1/I3/I4/I10/I14 训练；点云主线只允许 I6/I7 指向的单一修复进入训练。
4. **任何组合都延后**：I1+I11、I2+I14、I8/I17 融合/蒸馏都必须等各自单变量已过线后再做，且只作为系统级结果，不替代归因表。

### 4. 必须修正的文档口径

1. I3/I14 目前仍偏"目标侧 oracle"：最终版必须补"输入是否能预测 POD/势场系数"这一关，否则不能作为训练立项依据。
2. I18 的 Go/No-Go 必须改成"Pa p95 不退或提升"，不能写成"未见 case 的 Pa p95 获得提升"。
3. I20 的 Go 门槛建议分两级：`τ_θ`/global x/y R² >0.05 是观察 Go，>0.10 才是结构性 Go；magnitude 退化超过 0.005 直接 No-Go。
4. 推荐执行顺序应从"机会列表"改成"门控队列"：先 oracle，再单变量训练，再多 seed，再组合复测。现在文档已经足够发散，不需要继续加 I21 之后的新方案。

### 5. 最终建议给后续讨论会

后续讨论会不要再问"还有没有新招"，应围绕四个裁决问题收敛：

1. `τ_θ` 是否在现有数据中真的可学，还是信息论上接近不可学？
2. G4 展开链路是否已经稳定到可以承载 I1/I2/I14，还是 5507 只是局部过拟合假象？
3. early peak 是 backbone 特征被覆写、head 过拟合，还是 val 集小样本选择噪声？
4. 论文目标是否接受"病例级/区域级 surrogate 明显改善但点级 R² 只小幅提升"的叙事？

只有这四个问题有明确答案后，才值得决定下一次 GPU 训练。

## 2. I1 · θ-周期结构先验 + θ-roll 对称增强（P0 · G4 第一增强）

**科学依据**：展开后的壁面 WSS 场是圆柱面 S¹×[0,1] 上的函数，θ 维**数学上严格周期**。现 G4-a 首版 2D U-Net 若用零填充，会在 θ=0/2π 处人为制造边界伪影；且 θ 参考系（"切向均值 + 垂直平面投影"）对每个 case 本质任意——把整张网格沿 θ 循环平移 k 格，输入通道（几何、条件）与标签（WSS）**同步平移后仍是合法样本**。这是精确群对称（循环群 C_n），不是近似增强，无标签噪声风险。小样本（train ~60 case）下精确对称增强是文献中最稳的免费收益。

**改动阶梯（单变量、分表）**：

1. `I1-a`：U-Net 所有卷积 θ 维改 **circular padding**（s 维仍零填充）。仅结构先验，不动数据。
2. `I1-b`：在 I1-a 上加 **train-only θ-roll 增强**（每 graph 每 epoch 随机循环平移整数格；val/test 不增强）。
3. `I1-c`（可选）：测试期 θ-roll TTA 平均（8 个相位预测取均值），纯推理侧。

**前置（0 成本）**：代码审计确认展开链路中没有任何特征隐式依赖 θ 绝对原点（如 θ 本身作为输入通道——若有，改为 sin/cos 编码或随 roll 同步平移）。

**Go / No-Go**：对照同 seed G4-a plain。I1-a Go：Δ≥+0.01 或 val-test gap 明显缩小；I1-b Go：累计 Δ≥+0.02。两档都在 ±0.005 内 → 维持 plain，但 circular padding 因零成本可默认保留。

**风险**：若 G4-a 的 θ 网格只有 n_sectors=4，roll 增强名义状态只有 4 个，收益受限 → 与 n_sectors=8 网格联动测试（n_sectors 变更须新表）。

## 3. I2 · 网格内禀标架矢量预测（P0 · 攻 `wss_x/y`≈0 的第三条路）

**科学依据**：`wss_x/y` 失败的根因假设 R1（坐标系负担）经 G1 等变路线检验失败（5441：VN 头令 z 都崩），但 G1 失败否定的是"用等变算子学标架"，**不否定 R1 本身**。G4 展开网格的每个 cell 天然携带内禀标架：e_s（流向切向）、e_θ（周向切向）、n（法向）——由几何**确定性构造**，无需网络学习，且跨 case 语义一致（流向就是流向）。在此标架下监督 (τ_s, τ_θ)（τ_n≈0 由物理保证），把"学旋转对齐"彻底从任务中移除。这正是 H11（流向-跨流向标架）的 G4 具体化：H11 停在观察池是因为点云上中心线投影标架构造不稳，而 2D 网格把标架构造变成了展开链路的副产品。

**与 TODO-5 No-Go 的区别（防翻旧账质疑）**：TODO-5 `local_v1`（5146）是**点云 kNN 切平面标架**——逐点估计、噪声大、跨 case 不一致；本方案标架来自中心线 + 展开几何，光滑且全局一致。两者仅共享"局部坐标"四个字，须在立项书中显式声明此区别。

**前置 oracle I2-o（0 重训 · 必做）**：

- 把 GT WSS 投影到网格标架，得 (τ_s, τ_θ) 2D 场；统计跨 case 分布方差 vs global (wss_x, wss_y)；
- 对 τ_θ（跨流向，对应二次流）做 20/50 模态 POD 重建 R² 与线性/GBDT probe（与 F0 `pod_2d` 同口径，分表）；
- **过线判据**：τ_θ 的 probe/POD 可学性显著高于 global x/y（参照 G0-a 门槛 +0.1），且 τ_n 残差占比 <10%（验证标架与标签一致性，否则先回 Q3 标签 QA）。

**训练改动**：G4 U-Net 输出 2 通道 (τ_s, τ_θ)，反映射时经标架旋转还原 global xyz 再算标准指标——**评估口径不变**，与历史可比。

**Go**：τ_θ 还原后等效 `wss_x` 或 `wss_y` test R² **>0.10**（历史从未达到，破线即本项目首个矢量结构性证据）；且 magnitude R² 不退（Δ≥−0.005）。**No-Go**：τ_θ 仍 ≈0 → 与 H14 联动，转入"跨流向分量信息论不可学"的论证线（CFD 入口微扰实验），改写论文叙事而非继续刷分。

## 4. I3 · POD 低秩系数头 + 残差场分治（P1 · 复活 TODO-52）

**科学依据**：F0 `pod_2d` 已证 20 个 train-only POD 模态可重建 test WSS 场至 R² 0.672——WSS 场在 2D 表示下**本质低秩**。直接逐 cell 回归让网络同时承担低频主结构与高频残差；分治式目标分解（低秩系数 + 残差场）在算子学习文献中是标准降方差手段。TODO-52 此前标"候选（须 2D oracle 复核）"——**oracle 现已过线**，立项条件成熟。

**结构**：共享 2D encoder → ① 全局池化头回归 20 维 POD 系数 → 重建低秩场 WSS_lr；② 卷积解码头预测残差 ΔW；输出 WSS = WSS_lr + ΔW，端到端训练（系数头可加辅助 loss，权重单变量扫描禁止——只取一档 0.1）。

**Split-safe**：POD 基底**只在 train 集**拟合（与 F0 oracle 一致）；基底固定后 val/test 仅投影。基底文件入 manifest，禁止用 test 统计更新。

**Go**：vs 同 seed G4-a（或 I1 最优版）Δ≥+0.03；或 Δ≥+0.015 且病例 Pa p95 ≥ 0.347 门槛。**No-Go**：±0.005 内 → 封存，结论回填 TODO-52。

**风险**：低秩头可能"吃掉"全部容量使残差头退化为 0 → 监控 ΔW 方差占比；若 <5% 说明退化成纯 POD 回归（已知上限 0.672 但泛化未证），需 ablation 报告两头各自贡献。

## 5. I4 · θ-周期谱算子换型（P1 · I1 之后的架构升级）

**科学依据**：FNO 系算子的频域卷积在**周期维上精确**（无截断伪影）——(s, θ) 网格的 θ 维正是周期的，这使 Geo-FNO/FNO 在此问题上比通用 2D U-Net 有更贴合的归纳偏置；s 维非周期，用标准 padding-FNO 或 s 维保留卷积的混合架构（θ 谱 + s 空域）。TODO-48 此前 P3 搁置是因为点云上无规则网格；G4 链路建成后这个前提障碍已消除。

**纪律**：仅在 I1 完成后立项（先把 U-Net 线的周期先验收益拿干净，否则架构与 padding 两个变量混淆）；新架构新表，对照"I1 最优 U-Net 版"而非 G4-a plain。

**Go**：Δ≥+0.02 vs I1 最优版，且 2D→3D 反映射损失 <0.02 不恶化。**No-Go** → U-Net 线收口，G4 架构维度封存。

## 7. I6 · 梯度冲突诊断 → p/WSS 解耦训练（P0 · 与 G4 并行 · 攻 S1 早峰之谜）

**科学依据**：5466/5468/5439 一致显示 `best_wss` 在 ep6–13 达峰后 `val_wss` 单调退化，而 `r2_p` 持续改善——多任务学习中的典型**梯度冲突 / 任务支配**症状（共享 backbone 上 ∇L_p 与 ∇L_wss 方向负相关，优化后期被样本量与信噪比占优的压力任务拖走）。此前 TODO-15（GradNorm）P3 搁置是"盲调"；本方案先用诊断把假设坐实，再对症下药——符合 G0 oracle-first 纪律。

**Phase 1 · 诊断（≈0 GPU）**：

- 在下一次任何计划内训练（如 G4 对照重跑或 I1 训练）中插桩：每 N step 记录共享参数上 cos(∇L_p, ∇L_wss)、两任务梯度范数比；
- 同时记录逐 epoch 的 val_wss / val_p 曲线对齐图。
- **判读**：若 ep>10 后 cos 中位数 <0 且范数比 >5 → 冲突假设成立，进 Phase 2；若 cos≥0 → 假设证伪，早峰归因转向 I7（特征衰减 vs head 过拟合），本线关闭。

**Phase 2 · 解耦（诊断过线才开 · 三选一按序）**：

1. `I6-a` **两阶段训练**：联合训练至 val_wss 峰值邻域（ep~10）→ 冻结 backbone → 仅 wss_head 低 lr 续训。最便宜，先做。
2. `I6-b` **PCGrad 式冲突投影**：冲突 step 把 ∇L_wss 投影到 ∇L_p 的正交补（或反向）。
3. `I6-c` **任务专属 BN/LayerNorm + 末两层分叉**：结构性解耦，最后做。

**Go**：vs post5463 band 均值 0.425，Δ≥+0.02 且 best_wss epoch 明显后移（>20）、`r2_p`≥0.85。**No-Go**：±0.005 内 → 封口，结论回填 TODO-15（升级其证据等级后关闭）。

**价值兜底**：即使 No-Go，诊断数据本身回答 Q7（checkpoint 选优假阳性）并可写进论文的 training dynamics 分析——零浪费。

## 8. I7 · checkpoint 特征衰减探针（0 GPU · I6 的姊妹诊断）

**做法**：取一次已完成训练的多 epoch checkpoint（若未保存，下次训练每 10 ep 存一份 backbone-only，开销可忽略），在**冻结特征**上对 test WSS 训练线性/GBDT probe（probe 只在 train 特征上拟合），画"probe R² vs epoch"曲线，与 val_wss 曲线对照。

**判读分支**：

- probe R² 与 val_wss 同步衰减 → **特征被覆写**（支持 I6 冲突假设 / I6-a 冻结策略）；
- probe R² 持平而 val_wss 衰减 → **head 过拟合**（解法转向 head 正则 / EMA 选优，衔接 TODO-20，而非动 backbone）；
- probe R² 全程低 → 特征本来就没编码 WSS 信息（支持换轨 G4 的决策，增强叙事）。

**产物**：`outputs/field/f0_decision/v3p_i7_ckpt_probe_<date>.json` + 曲线图；不出 Main 表。

## 9. I8 · 跨表示融合（P2 · 纯后处理 · G4 出结果后的免费收益）

**科学依据**：集成收益 ∝ 成员误差去相关程度。5258（同架构 deep ensemble）No-Go 的原因正是同表示误差高度相关（R²+0.005）；点云模型与 2D 展开模型的失败模式应当结构性不同（点云败在全局低频/坐标系，2D 败在分叉展开扭曲与反映射平滑）——这是与 5258 本质不同的命题，不属于重跑已 No-Go 配方。

**前置（0 成本 · 必做）**：在 val 上计算两模型逐点残差的 Pearson 相关；ρ<0.7 才有融合空间，≥0.85 直接放弃。

**做法**：逐点凸融合 ŷ = α·ŷ_pc + (1−α)·ŷ_2d，α 仅在 **val** 上网格搜索（步长 0.05），test 只报一次。升级版（残差相关性低时）：α 随 (s,θ) 位置变化的分区融合（如分叉邻域信任点云、平直段信任 2D），分区规则同样 val 内定。

**Go**：test Δ≥+0.015 vs 两个单模型中较优者，且 Pa p95 不退。**纪律**：融合结果**不替代母版**，作为"系统级最优"单独列报；两成员模型指标照常分列。

### 13.1 I11 · 镜像（宇称）精确增强（P0 · 全轨道通用 · 零标签噪声）

**科学依据**：Navier-Stokes 方程在空间反射下等变：对几何与边界条件做镜像 P，解场同步镜像——(Px, Pu, Pτ) 是**精确合法**的新样本，无任何标签近似。镜像翻转血管的手性（弯曲/螺旋方向），Dean 涡与二次流随之一致翻转，对网络是真正"没见过"的几何。train ~60 case 直接翻倍，这是继 I1 θ-roll 之后第二个零风险对称增强，且**点云主线与 G4 2D 轨道都适用**（2D 侧等价于 θ→−θ 镜像 + τ_θ 变号）。

**前置（0 成本 · 必做）**：列出全部输入/标签通道的宇称变换规则并审计——坐标/法向/切向/WSS 矢量按反射矩阵变换（x→−x 时 wss_x 变号），Curvature/Abscissa/NormRadius/压力为标量不变，赝矢量类特征（若有叉乘构造）须额外变号。**任一通道变换规则不明确则该通道先修复再开训**。

**改动**：train-only 随机镜像（p=0.5），val/test 不增强；单变量对照同 seed 无增强版。

**Go**：Δ≥+0.01 vs post5463 band（增强类收益预期温和）；val-test gap 缩小为辅助证据。**No-Go**：±0.005 内 → 关闭，但审计产物保留（服务 I16）。

**风险（诚实声明）**：真实解剖手性分布有偏（如主动脉弓旋向一致），镜像样本物理合法但解剖上 off-manifold——可能帮助泛化也可能引入分布偏移；由实验裁决，不预设结论。

### 13.2 I14 · 壁面 Helmholtz-Hodge 目标重参数化（P0 · 物理+算子 · 与 I2 配套）

**科学依据**：壁面切向矢量场（WSS）在 2D 流形上有精确的 Helmholtz-Hodge 分解 τ = ∇φ + J∇ψ（+ 低维调和分量），φ/ψ 是标量势：φ 捕捉源汇结构（分离/再附线，WSS 拓扑的临床要害），ψ 捕捉旋转结构（二次流印记）。预测两个**标量场**再微分重建矢量，有三重收益：①积分使目标光滑一阶（低频化，小样本更易学）；②矢量场拓扑一致性由构造保证；③在规范 (s,θ) 矩形域上 Fourier 基**跨 case 共享**——这正好绕开 H4 谱图卷积的"谱漂移"死结（H4 的问题是每个 case 的 Laplacian 基底不同；规范展开域上基底是同一个）。

**前置 oracle I14-o（0 重训）**：在 (s,θ) 网格上对 GT WSS 切向场做 FFT-Poisson 分解（θ 周期、s 端 Neumann，调和/均流分量单独记账）：报 ①重建闭合误差（应 <1%，否则展开/标架有问题，回 Q3）；②φ/ψ 的谱能量集中度 vs 原始 τ_s/τ_θ 分量——**过线判据**：φ/ψ 前 20 模态能量占比显著高于原分量（目标确实更低频）。

**训练改动**：G4 U-Net 输出 (φ, ψ) 两通道 + 可微分重建层 → 矢量；损失 = 重建矢量 MSE + 势场辅助 MSE（一档权重，禁扫描）。与 I2 天然兼容（重建直接给出 τ_s, τ_θ）。

**Go**：还原后 `wss_x` 或 `wss_y` test R² **>0.10**（与 I2 同结构性门槛），或 magnitude Δ≥+0.02。**风险**：对预测势场求导会放大高频误差——这是"目标更光滑"的代价，由重建矢量 MSE 主导损失来约束；若 oracle 显示 φ/ψ 并不更低频，直接不立项。

### 13.3 I10 · 近壁壳层张量 (s,θ,n)（P1 · 把边界算子写进输入结构）

**科学依据**：`WSS=μ·∂u_t/∂n|_wall` 的全部证据都在**法向剖面**里，而 kNN 球邻域采样既无序也不保证沿法向覆盖（R2 的根源）。在 G4 网格每个 cell 沿内法向取固定无量纲深度 {δ₁..δ_k}（以局部半径归一）插值速度/压力，得 (s, θ, n, C) 结构化壳层张量——边界层作为体数据直接进网络。首版把 n 维当通道维用 2D 卷积（便宜），升级版 3D 卷积。**与已 No-Go 线的区别**：5276/5302 败在把差分当**输出/监督**；I5-c 只给单个差分代理标量；本方案给**完整有序剖面**作输入，让网络自己学算子——三者命题不同。

**前置 oracle I10-o（0 重训 · 即 H3 oracle 的网格化执行）**：用 GT 速度在法向剖面上做有限差分 WSS 代理，报与 GT WSS 的 Spearman / R² / 插值失败率。**过线**：Spearman >0.6 且失败率 <10%（法向翻转问题先回 Q3）。

**Go**：Δ≥+0.03 vs 同期 G4 最优版。**风险**：内部场点在近壁的密度可能不足以支撑 k≥4 层插值 → oracle 阶段同时报每深度的插值覆盖率，覆盖差则减层或改抛物线拟合。**衔接**：H3 由观察池升级为本方案的 oracle 前置，结论双向回填。

### 13.4 I16 · 病例级几何规范化（P1 · 几何 · 数据侧的"廉价等变"）

**科学依据**：G1 在架构层消除位姿滋扰失败（等变算子难训），但同一目标可以在**数据层**用确定性预处理达成：每 case 做全样本刚体规范化（入口截面中心置原点、入口法向对齐 +z、中心线主平面对齐 xz），再以局部半径 R(s) 无量纲化径向坐标。整个样本（坐标+矢量标签+法切向）做同一个刚体变换——**精确变换，零标签噪声**；评估时变换回原系，口径不变。**与 TODO-5 No-Go 的区别**：TODO-5 是逐点局部标架（噪声大、不连续）；本方案是每 case 一个全局刚体变换 + 光滑径向缩放，性质完全不同。

**前置（0 重训）**：规范化前后分别跑几何特征 → GT WSS 的 GBDT probe；规范化后 probe R² 提升 ≥0.05 才开训（证明位姿确实是当前滋扰项之一）。

**Go**：Δ≥+0.015 vs band；`wss_x/y` 出现非零信号为强加分项。**No-Go** → 与 G0-a 结论合并，进一步坐实"坐标系不是主瓶颈"，反向支持 H14 论证线。**成本**：预处理脚本 + 重建图（复用 5463 renorm+regraph 链路），新 graphs 新表。

### 13.5 I13 · DeepONet 边界算子分解（P1 · 算子学习的另一极）

**科学依据**：FNO（I4）把问题当"场到场"映射；DeepONet 把它当**算子求值**：branch 网编码 case 级条件（轻量几何 encoder 的全局 latent + BC/Re 标量），trunk 网编码查询点（(s,θ) + 内禀几何特征），输出为两者内积。优势：①trunk 在**所有 case 的所有点**上共享——点级有效样本数 ~10⁶ 量级，对 60-case 小样本是结构性利好；②算子学习的万能逼近定理给出理论支撑；③天然支持任意查询点，**无 2D→3D 反映射损失**（直接在原壁面点上求值，绕开 G4 §9 风险 3）。

**改动**：独立模型线 `V3P-I-13-DeepONet`，对照同 split G4 最优版与点云基线双表。branch latent 维度刻意压小（≤64）防 case 记忆。

**Go**：Δ≥+0.02 vs 两个对照中较优者。**风险**：branch 退化为 case-ID 查表 → 用 val gap 与 latent 插值平滑性检查（同 H9 的反泄漏纪律）。

### 13.6 I9 · 序列化注意力点云骨干（P2 · 架构 · 攻长程感受野）

**科学依据**：二次流由上游几何历史决定（Q8/H12 的核心疑点），kNN 局部池化的球状感受野在信息论上可能根本拿不到上游证据。序列化注意力骨干（Point Transformer V3 一族：空间填充曲线序列化 + 窗口注意力）以近线性代价给出长程感受野，是与"等变"正交的另一种骨干升级——**G1-b 封掉的是等变骨干，非等变强骨干从未试过**。

**前置（≈0 GPU · 即 H12 的归因检查）**：对 post5463 基线做高 WSS 点的梯度归因/感受野分析；若归因质量已集中覆盖上游中心线方向（感受野不缺）→ 本方案直接不立项，省一次换骨干。

**Go**：三 seed Δ≥+0.03 且 `r2_p`≥0.85。**风险（诚实）**：注意力骨干在 60 case 上极易过拟合，且换骨干是大改动——故排 P2，仅在 G4 轨道与 I6 解耦都收口后、且归因诊断显示感受野确实不足时启动。**衔接**：H12 的"Centerline-Attention 宏观通道"是本方案的轻量替代，归因诊断结果同时服务两者的取舍。

### 13.7 I17 · 跨表示蒸馏（P2 · 把 I8 融合收益内化）

**科学依据**：若 I8 证实点云与 2D 模型误差去相关且融合有效，下一步是把融合收益**内化**：在共享壁面点上加互蒸馏一致性损失（各自预测互为软目标，按各自 val 误差/不确定性加权），得到单模型部署形态。比 I8 多的增益来自训练期的正则效应（一致性约束抑制各自的表示特异性过拟合）。

**前置**：I8 的残差相关性检查（ρ<0.7）+ I8 融合本身 Go。**Go**：单模型 Δ≥+0.015 vs 各自基线。**风险**：teacher 错误传播 → 不确定性加权（衔接 H8）+ 一致性权重单档不扫描。

### 13.8 I15 · 内禀谱几何特征通道（P2 · 几何 · H2 的描述子扩展）

**科学依据**：热核签名（HKS）、到入口/出口/分叉的测地距离场、局部截面傅里叶形状系数等**内禀（位姿不变）**几何描述子，提供 kNN 邻域拿不到的中尺度几何上下文，且预计算便宜、对点云与 2D 网格通吃。与 H2（branch_id / 拓扑坐标）互补：H2 是离散拓扑标签，本方案是连续谱几何场。

**前置 probe（0 重训 · 与 H2 同协议合并执行）**：GBDT probe 现有特征 vs +内禀描述子，看 top-k AUC / p95 Pearson / 局部分量 R² 增量；有显著增益才进训练（每描述子族单变量）。**Go**：Δ≥+0.01 且病例级不退。**衔接**：probe 结果与 H2 互填，避免两次跑同一协议。

### 13.9 I12 · 物理一致幅值缩放增强（P3 · 弱候选 · 显式标签噪声声明）

**科学依据（弱）**：量纲分析下同几何、流量缩放 k 的解在 Stokes 极限严格满足 WSS→kWSS；有限 Re 下非线性破坏严格性，只在 k≈1 邻域近似成立。作为增强：对 (流场输入, WSS 标签) 做联合小幅缩放（k∈[0.9,1.1]）——**这是有标签噪声的近似增强**，与 I1/I11 的精确对称有本质区别，故列 P3。

**前置 oracle（0 重训）**：跨 case 回归 log p95(WSS) ~ log U_in，报经验幂指数与拟合 R²；R²<0.5（缩放律在数据中不稳）→ 直接不立项。**Go**：Δ≥+0.01 且 val-test gap 不恶化。**纪律**：与 TODO-28（FFD 几何增强）同属"近似增强"类，禁止与精确增强（I1/I11）在同一 ablation 表内混报。

## 14. 第三批发散方案（AI 补充讨论 · 2026-06-11）

> 以下是由 AI 补充的三个发散性思路，侧重于**目标函数物理约束**、**训练动力学**与**损失加权**，作为后续讨论的增补备选。纪律与前批次相同：oracle-first、单变量、新表。

### 14.1 I18 · 壁面局部 PDE 残差正则化（PINN 式物理守恒约束）
**维度**：物理+优化
**科学依据**：流体力学中，壁面临近区域的动量守恒要求壁面切应力（WSS）的散度、近壁速度的二阶导与沿面压力梯度 $\nabla p$ 紧密平衡。目前模型仅以纯数据驱动（L2 Loss）回归 WSS，预测的矢量场可能不满足流体动量方程的局部平衡。如果在损失函数中引入弱形式的物理约束项（如 $|\nabla \cdot \tau_{pred} - f(\nabla p)|$ 的残差），相当于给网络提供了一个不完全依赖真实标签的无监督结构梯度信号。
**前置 oracle（0 GPU）**：在 2D $(s, \theta)$ 网格上列出近似的壁面动量平衡方程，计算 GT WSS 场在该方程下的残差，确立物理 loss 的理论下界。若 GT 残差极大，说明数据本身不满足该简化方程，则本思路不可行。
**Go / No-Go**：加入物理 loss 后，`wss_r2_wss` $\Delta \ge +0.015$，且病例级 Pa p95 不退或获得提升（物理约束若有效，应至少不损害 OOD / 病例级泛化）。

### 14.2 I19 · 基于几何复杂度（曲率/分叉）的课程学习（Curriculum Learning）
**维度**：训练动力学
**科学依据**：当前训练为全数据 shuffle，网络在初期同时面临极其复杂的二次流（极端分叉、剧烈弯曲）与简单平直流，导致 S1 中 early peak 和后续退化（网络学不会复杂部分，却把简单部分的特征破坏了）。人类学习流体也是从直管到弯管再到分叉管。如果在低曲率、无分叉（或少分叉）的样本或局部区域上预训练，再逐步解冻/混入复杂几何，可能缓解任务冲突与退化。
**前置（0 GPU）**：量化所有 case 的“几何复杂度得分”（根据平均曲率、中心线扭率、分叉截面积比等），确立划分阈值。
**做法**：将 train 集切分为 3 个难度梯队（Easy, Med, Hard），采用阶段式调度策略（先 Easy，再 Easy+Med，最后 Full）。
**Go**：相比基线，val_wss 曲线的衰减被明显抑制，且 Hard 样本组上的 R² 获得结构性提升（$\Delta \ge +0.03$）。

### 14.3 I20 · 跨流向（二次流）放大重采样与加权（Spatial Focal Loss）
**维度**：目标+数据增强
**科学依据**：`wss_x/y` 极难预测的本质原因可能是二次流导致的跨流向 WSS 分量（$\tau_\theta$）在全局 L2 Loss 中被主导流向分量（$\tau_s$）淹没。模型倾向于牺牲小幅度、高频的特征来保证全局 MSE 最小（类似目标检测中的类别不平衡）。针对 GT 中 $\tau_\theta$ 幅值占主导或反常（例如二次流分离区）的区域，动态调高 Loss 权重。
**前置（0 GPU）**：统计 train 集全场像素的 $\tau_s$ 和 $\tau_\theta$ 能量比分布，定义“二次流主导区”（如 $\tau_\theta / |\tau| > 0.3$ 的 cell）。
**做法**：引入 Spatial Focal Loss，赋予二次流主导区 3~5 倍的梯度权重。
**Go**：$\tau_\theta$ 还原后的 global x/y 预测能力破冰（R² > 0.05），且整体 WSS magnitude 不崩溃。

### 15.2 本轮新发现的三处技术问题（前两轮均未捕获）

1. **I1-b θ-roll 与 I3 POD 系数头互斥**。POD 基底依赖跨 case 的（近似）θ 对齐；θ-roll 增强恰好破坏这种对齐——roll 后样本的 POD 系数被结构性搅乱，二者**不得同训**（I11 的 2D 镜像 θ→−θ 同理）。而 **I14 的 FFT-Poisson 分解对 θ 平移等变**（周期维上 Poisson 求解与平移可交换），与 roll/mirror 增强完全兼容。**推论**：若低秩目标线立项，**兼容性上 I14 优于 I3**；I3 只能与 I1-a（仅 padding）共存。此互斥关系写入 I1/I3/I11/I14 的组合纪律，θ 审计清单加一项"roll × POD 兼容性检查"。
2. **I20 缺与 TODO-42（5253 No-Go）的衔接**——这是 I20 最近的先例败绩：区域加权类 loss 混入主训练导致全局退化（best_wss 0.378，−0.021）。I20 命题确实不同（分量方向不平衡 vs 幅值排序），但必须：①立项书显式区分；②设 magnitude 硬防线（Δ≤−0.005 立即停，比通用 ±0.005 更严）；③**只在 I2/I14 目标已换的前提下做**——在 global xyz 目标上给"二次流主导区"加权，权重打在被坐标系拆散的分量上，大概率重演 5253。
3. **执行口径冲突**。增补两轮后文档同时存在三套优先级（§10、§13.10、评审 C 表），且互相矛盾（如 I11/I16 在 §13.0 为 P0/P1 首波，在终审结论中为暂缓）。已用本节统一，旧表标注"仅留作历史"。

### 15.5 Go / No-Go 总表修订（在 §11 基础上叠加）

1. **矢量分量两级门槛**（采纳前轮）：还原后 `wss_x` 或 `wss_y` test R² **>0.05 = 观察 Go**（补 seed）、**>0.10 = 结构性 Go**；且**新增必要条件**：中位方向角误差较基线下降 + 高 WSS top-k Dice 不退——排除"坐标变换搬运 R²"的假突破。
2. **I18 口径修正**（采纳）：病例级判据为"Pa p95 **不退或提升**"。
3. **增强类（I1-b / I11 / I12）**：Go 判读必须附 val-test gap 变化；gap 扩大即使点级 Δ 达标也降级为观察。

5. **I20 专属防线**：magnitude Δ≤−0.005 立即 No-Go（严于通用门槛，理由见 §15.2-2）。
