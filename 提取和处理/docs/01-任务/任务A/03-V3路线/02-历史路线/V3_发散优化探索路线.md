## 1. 根因再压缩（2026-05-28）

路径 A（local / MagOnly / MagCons）与路径 B 首跳（GeomB1 / VelGradB1）均 **No-Go** 后，可得到比「继续调权重」更硬的结论：

| 已被排除 | 留下的真正约束 |
| --- | --- |
| 容量不足、loss 权重、4 维冗余、WSS 表示改造（global） | **(a)** `wss_x/y` 二次流对局部几何 + 入口微扰极敏感，60–179 case 在标准监督下难泛化 |
| `dist_to_wall` / 法向梯度上下文 / 单档 kNN pool 微调 | **(b)** 无「几何相似病例软共享」或「入口波形条件化」机制，每 case 靠隐式记忆 |
| 单点 OOD、val 语义、三域 train-only 归一化（主要救 P） | **(c)** 目标以瞬时点级 R² 为主，**WSS = μ·(∂v/∂n)\|_wall** 的边界算子结构未写入监督 |

**推论**：在「同一监督 + 同一 FieldPointNeXt local-pool + 60 case」空间内继续单变量，**结构性突破概率低**。须至少改动 (a)(b)(c) 之一。

**优化后的主判断（2026-05-29 · F0 证据落地后修正）**：

- **不再优先**做“再加一个 loss / 再换一个小 head / 再加轻微正则”的实验；这些更可能落在 `0.39–0.40` 噪声带。
- ~~优先验证尺度与入口条件（TODO-31 / 37）~~ → **F0 残差分层显示 fast/slow ΔR²≈0.03、BC Δ≈0.02 均未达 0.08 触发线 → TODO-31/37 后置**，不作为下一批 Probe（详见 §4 阶段 0 结论与 §10）。
- **当前下一跳（2026-06-02）**：**PC-v2/BLContext 正式 Probe**；**5302/5276 direct vel_diff 关闭**；PGradFeat 未升母版；路径 C **TODO-3/1**；维持 **4957**。
- **oracle 复核优先于关闭**：TODO-30/33 的 GT 近壁差分 oracle 给出 **mean Pearson −0.34**，但该 oracle **未投影切向、最近内部点非沿法向、Δn 为欧氏距离且受近壁网格各向异性影响** → 负相关大概率是 **oracle 噪声**而非物理不成立；应**先修正 oracle（切向梯度 ∂u_t/∂n + 沿法向采样）再判 30/33**，当前记为「暂缓核查」而非 No-Go。

**确认档（27b）— 完整 5-fold CV（勿在探索期默认启动）**

- **做法**：`split_AG_v1` train 60 case → 5-fold，每 fold **48 train / 12 val**，**每折从头训练**同一配方（如 AsymW-a 或已定 Go 的 `V3P-F-29-EGNN`）。
