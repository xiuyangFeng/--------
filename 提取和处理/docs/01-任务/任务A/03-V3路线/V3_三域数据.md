# 第一部分 · 诊断

## 1. 摘要

三域实验（**V3D**）相对 **V3P**（仅 `split_AG_v1`）的对比须分 **pre-fix**（4771–4774，4664 口径）与 **post-4901**（`split_data_new_v3_v3`，train **179**）两张表：

| 探针 | V3D pre-fix | **V3D post-4901** | V3P（AG-only） |
| --- | ---: | ---: | ---: |
| WSS `wss_r2_wss` | 0.255 | **0.243** | **0.394±0.005**（AsymW 三 seed；Main-PW **0.365**） |
| 压力 `r2_p` | −4.45 | **0.984** | 0.962 |
| 速度 `r2_vel_mag` | −1.03 | 待 Probe-V `summary` | 0.294 |

**核心结论（诊断仍成立）**：pre-fix 退步主因是 **`normalize.py` 非严格 train-only** + 跨域尺度异构 + 训练离群；**4901 后压力探针已恢复可学**（Gate-P ✅），WSS 标量**未抬升**（仍在 **~0.24–0.26**）。

**与 V3P 禁止混表**；分域 test（AAA/AG/ILO）见阶段 3 `eval_by_domain_test/metrics_by_domain.json`。

## 4. 根因分析：为何数据多了反而变差

### 4.1 非 V3P 式单点 OOD

- 259 例图 z-score 扫描：**0/53 test** 超过 train 各指标 max 的 **1.5×**。
- `AG/fast/PENG_JI_MING` 已在 denylist，**不在** `split_data_new_v3`。
- test 集 **无** `inlet_v < 1e-8` 病例。

### 4.2 `normalize.py` train 匹配缺陷（P0，**代码已修 2026-05-19**）

**修复前**：`collect_global_statistics` 用叶目录名 `after`/`before` 碰撞 → 意图 train **181**、实际 **226**（+45，含 test 22 / val 11）。

**修复后**：

| split 版本 | train | Gate-0（`verify_train_stats_dirs`） |
| --- | ---: | --- |
| v2（2026-05-19） | 180 | stats_dirs=180，extra=0 |
| **v3（2026-05-20）** | **179** | stats_dirs=**179**，extra=0（剔除 SU_KAI_LI + `ILO/GUO_AI_JUN-0/after`） |

作业 **4664** 产出作废。作业 **4901**（**2026-05-21** 完成）train-only **179** 重算并应用：`p.std≈**3939**`（4664 ~**10320**），`wss.std≈3.14`；normalize **267/277** + 转图 **267/277**；**179 train** 与 **257** 候选池 `normalized`/`graphs` **0 缺失**。

→ 旧参数与 Probe-P **val≈0.97 / test≈−4.45** 相符（口径错位）；**须重跑** V3D-Probe-P / Probe-WSS（**勿与 4771–4774 混表**）。

### 4.3 跨域异构 + 训练离群污染

- AAA / ILO 壁面压力、WSS 分布与 AG 差异大；**单一全局 z-score** 难以同时适配三域。
- **`AAA/ruputer/SU_KAI_LI`**（train）`p_mean≈146406 Pa`、`wss_max≈156`，拉高全局 `p.std≈10320`、`wss.std≈4.37`（**未在 denylist**），危害类似 V3P 的 PENG，但位于 **train**。
- **`ILO/GUO_AI_JUN-0/{after,before}`**：已入 denylist 并从 split v3 train 移除（Fluent 未完成）。  
- **`DONG_KE_QIN-0/{after,before}`**：壁面 **81/81**；步骤 2–3 已刷新（**4899/4900**），纳入 train-only 统计 **179** 例。

### 4.4 与 V3P 对比

| 维度 | V3P | V3D |
| --- | --- | --- |
| 数据域 | 仅 AG | AAA + AG + ILO |
| 主要失败模式 | 1 例 test OOD | 系统性口径 + 域偏移 + 分散离群 |
| 压力探针 | 可学（排 PENG 后 R²≈0.96） | post-4901 **R²≈0.98**（4773 pre-fix 作废） |
| WSS 探针带宽 | ~0.36–0.40（AsymW **0.394±0.005** 三 seed） | post-4901 **~0.24**（pre-fix 0.255） |
