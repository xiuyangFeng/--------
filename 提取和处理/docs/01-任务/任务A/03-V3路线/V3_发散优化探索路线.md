## 1. 根因再压缩（2026-05-28）

路径 A（local / MagOnly / MagCons）与路径 B 首跳（GeomB1 / VelGradB1）均 **No-Go** 后，可得到比「继续调权重」更硬的结论：

| 已被排除 | 留下的真正约束 |
| --- | --- |
| 容量不足、loss 权重、4 维冗余、WSS 表示改造（global） | **(a)** `wss_x/y` 二次流对局部几何 + 入口微扰极敏感，60–179 case 在标准监督下难泛化 |
| `dist_to_wall` / 法向梯度上下文 / 单档 kNN pool 微调 | **(b)** 无「几何相似病例软共享」或「入口波形条件化」机制，每 case 靠隐式记忆 |
| 单点 OOD、val 语义、三域 train-only 归一化（主要救 P） | **(c)** 目标以瞬时点级 R² 为主，**WSS = μ·(∂v/∂n)\|_wall** 的边界算子结构未写入监督 |

**推论**：在「同一监督 + 同一 FieldPointNeXt local-pool + 60 case」空间内继续单变量，**结构性突破概率低**。须至少改动 (a)(b)(c) 之一。

**优化后的主判断**：

- **不再优先**做“再加一个 loss / 再换一个小 head / 再加轻微正则”的实验；这些更可能落在 `0.39–0.40` 噪声带。
- **优先**验证尺度与入口条件是否被模型正确看到：TODO-31（Cf + Re/Wo）和 TODO-37（FiLM 入口调制）比 FFD/大 backbone 更适合作为下一批 Probe。
- **先做 oracle 再训练**：TODO-30/33 必须先看真实近壁速度差分能否解释 CFD WSS；TODO-52 必须先看 POD 低秩重建上限；TODO-28 必须先说明标签是否可信。

> **原则**：探索期 **不重训 5-fold**；先回填在跑/已提交实验，再做 **F0 探索档**（27a + 53 + 32 + oracle，≈0 GPU）→ 按证据选择 **1 条**低成本结构性 Probe（31/37/42/29）→ 条件性数据/物理分支（28/30/35）→ 定方向后再 **3 seed Main**；**确认档 5-fold（27b）仅论文/终稿需要时做。

- GT 近壁差分与 CFD WSS 若相关性低或符号/量纲不稳，TODO-30/33 不进入训练；若相关性高，再做差分推理或边界层斜率辅助。
- POD 10–20 模态若可重建主要 WSS 场，TODO-52 可升级为换轨候选；若重建误差上限过低，POD 回归降级。
- 残差若按入口 BC / domain / 几何簇有系统偏差，优先 TODO-37/38/19；若残差无结构，优先 TODO-31/29 或转临床区域叙事。

**确认档（27b）— 完整 5-fold CV（勿在探索期默认启动）**

- **做法**：`split_AG_v1` train 60 case → 5-fold，每 fold **48 train / 12 val**，**每折从头训练**同一配方（如 AsymW-a 或已定 Go 的 `V3P-F-29-EGNN`）。
- **成本**：≈ **5 ×** 单次 Main（200 ep 量级）→ 与「再跑 5 条主线」相当。
- **不适用**：仅用旧 ckpt 在 5 份 test 上算分（**不能**代替重训 CV）。
- **脚本建议**：`training/scripts/cv_wss_bandwidth.py`（新建）；输出 `cv_noise_floor_<date>.json`；**标签** `cv5_<recipe>_<date>`，禁止与单 split 0.399 混表。

**探索档 vs 确认档 — 能回答的问题**

| 问题 | 探索档够吗？ | 确认档 |
| --- | --- | --- |
| 初始化噪声多大？ | ✅ 三 seed | 可选 |
| 单次 test R² 置信区间？ | ✅ bootstrap | — |
| **换 split 后带宽多少？** | ❌ 粗估 | ✅ 5-fold |
| 是否接近 irreducible noise？ | ✅ 多 t 方差 + 病例级 | 辅助 |

## 6. Go / No-Go 模板（路径 F）

> 复制到 §10 每条实验记录。对照默认：**AsymW-a 4957** `wss_r2_wss=0.399`（seed1）或三 seed **0.394±0.005**。

| 层级 | Go | No-Go | 强 No-Go |
| --- | --- | --- | --- |
| 点级 | test `wss_r2_wss` **> 0.42** 且 Δ **≥ +0.03** | 在 ±0.005 噪声内 | Δ **≤ −0.02** |
| 分量 | `wss_x` 或 `wss_y` test R² **> 0.10**（新框架下） | 仍 ≈ 0 | val↑ test↓ 过拟合加剧 |
| 病例级 | Pa p95 Pearson **≥ 4957 − 0.05** 且 **> Probe 3645** | 点级升、Pa 降 | — |
| 区域级 | top-k Dice / IoU / AUC 至少一项明确优于 4957，且点级不明显降 | 区域升但病例/点级明显崩 | — |
| 前置证据 | FFD 标签 / GT 差分 / POD oracle 通过后才训练对应分支 | oracle 不成立仍开训练 | — |
| 口径有效性 | split / 归一化 / frame / target / 条件特征均有新标签；条件特征推理时可得且无目标泄漏 | 记录缺失导致无法复现 | 混表、target leakage、或 test-only 信息进入训练 |
| 决策类（27a） | 探索档结论写入 §10：seed/bootstrap/病例级/noise floor | — | — |
| 决策类（27b） | 5-fold mean ± std；**仅**终稿配方 | 探索期误开 5-fold | — |

**论文叙事**：点级 Go 不足时，若 TODO-53 区域指标 Go，仍可宣称 **临床区域 surrogate**（见精度突破路径 §10）。
