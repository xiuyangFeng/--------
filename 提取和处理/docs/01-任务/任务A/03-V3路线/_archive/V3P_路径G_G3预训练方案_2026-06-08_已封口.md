## 1. 立项依据

| 证据 | 结论 |
| --- | --- |
| G0-a No-Go（Δ=−0.022）+ G1-a0 封口（5441） | 坐标系负担**不是**主瓶颈；等变 alone 无法救 `wss_x/y` |
| G2-b 三 seed 同 split No-Go（0.404±0.012 < 基线 0.434） | 压力 kernel-attention **无稳健增益** |
| F0 noise floor 三 seed **0.394±0.005** | 在现表示空间内点级标量已收敛 |
| G3 几何盘点 **81/81** STL+merged+graphs | **数据侧**具备 SSL 预训练条件 |
| 路径 G §6 / Q9 | 预训练是攻 R3 的结构性手段，但必须 **split-safe** |

**命题（可证伪）**：在**不改变** FieldPointNeXt backbone 架构的前提下，流态感知几何 SSL 预训练能让微调后的 encoder 提取更贴近 WSS 物理的表征，使 test `wss_r2_wss` 相对同架构无预训练 **Δ≥+0.03**，且 `wss_x/y` 或病例级 Pa 至少一项不退化。

**实验结论**：命题 **No-Go** — SSL 预训练本身可收敛，但微调未达 +0.03 门槛。
